# Финальная работа курса «Старт в DevOps: системное администрирование для начинающих»
Закрепление теоретического материала и отработка подходов к решению задач, актуальных для специалистов профессии DevOps.

## Суть задачи
Задача максимально приближена к реальной: вам предстоит разработать инфраструктуру для централизованного управления учётными записями.

## Формат выполнения
Финальная работа состоит из шести задач, которые необходимо выполнять строго последовательно:

После завершения каждой задачи обязательно отправляйте её на проверку куратору.
Только после того, как куратор проверит и согласует решение, можно переходить к следующей задаче.
Такой подход позволит вам закрепить навыки поэтапно и получить качественную обратную связь на каждом шаге.

## Задачи финальной работы  
- Развёртывание удостоверяющего центра для выдачи сертификатов. 
- Создание и настройка VPN-сервера.  
- Настройка мониторинга.  
- Создание системы резервного копирования.  
- Подготовка документации.  
- План развития инфраструктуры.  

# Реализовано  
На ВМ клиента создается пара ключей fquest (закрытый ключ остается на машине клиента) и fquest.pub (публичный ключ - передается на серверную часть)  

В Yandex Cloud развернута ВМ на базе Ubuntu 2404. На данном ресурсе разворачивается серверная часть.  
FQuest.txt содержит скрипт создания ВМ в YC.  
Коннект к ВМ выполняется посредством SSH вида: ssh -l fquest 158.160.ХХХ.ХХХ (строка приводится YC в настройках ВМ)  

fquest.sh - скрипт создания и настройки OpenVPN сервера. Выполняенся на YC ВМ.  
Функции:  
__InstallOVPN_srv__ - утсановка необходимых сервисов и библиотек. В настоящий момент:  
  1. openvpn  
  2. easy-rsa  
  3. iptables  
  4. iptables-persistent  


__CustomSertCenter__ - настройка сертификационного центра.  
__GetServerSertificate__ - создает ключи/сертификаты: ca.key, ca.crt, dh.pem, crl.pem, server.crt, server.key, ta.key.  
__GetServerConfigFile__ - создает конфигурационный файл сервера в заданной дирректории.  
__StartOVPN__ - стартует сервис OpenVPN на созданном конфиге.  
__ConfiguringFirewall__ - создает правила-ограничения фаервола.
Помимо этого выполняет включение __ip_forwarding__ (не выделено в отдельную функцию).
